What ClaudioScribe Does

ClaudioScribe is a self-hosted audio processing tool that transcribes recordings from Plaud devices and creates structured documents using AI. It runs entirely on your own infrastructure.

Data We Access

When you connect Google Drive, ClaudioScribe requests the drive.file scope, which only allows access to files and folders that ClaudioScribe itself creates. It cannot read, modify, or delete any other files on your Google Drive.

Specifically, ClaudioScribe creates:

  • A ClaudioScribe/Documents/ folder for generated .docx summaries
  • A ClaudioScribe/Recordings/ folder for original audio files

Data We Store

ClaudioScribe stores the following on your local machine (in the config/ directory):

  • Google OAuth access token and refresh token (to upload files without re-authenticating)
  • Google Drive folder IDs (to know where to upload)
  • Your Plaud API token and Anthropic API key (for transcription and document creation)

All credentials are stored locally in a JSON configuration file on your machine or server. They are never transmitted to any third party.

Third-Party Services

ClaudioScribe interacts with the following services on your behalf:

  • Google Drive API — to upload documents and recordings to your Drive
  • Plaud API — to download your audio recordings
  • Anthropic API — to process transcripts into structured documents (transcript text is sent to Claude)

Data Sharing

ClaudioScribe does not collect, share, or sell any personal data. There is no analytics, telemetry, or tracking. The application runs entirely on your own infrastructure.

Data Retention

You control all data. Disconnecting Google Drive from the settings page immediately deletes all stored tokens. Files already uploaded to Google Drive remain in your Drive until you delete them.

Revoking Access

You can revoke ClaudioScribe's access to Google Drive at any time:

  • Click "Disconnect Google Drive" on the settings page, or
  • Visit Google Account Permissions and remove ClaudioScribe

Contact

ClaudioScribe is an open-source project. For questions or concerns, open an issue on the project's GitHub repository.
